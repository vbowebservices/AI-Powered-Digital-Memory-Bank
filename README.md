# AI-Powered Digital Memory Bank

## Overview
This is a simple Flask-based MVP for storing personal memories in text, image, or video form.

## Features
- User authentication (signup/login using email + password)
- Upload and view memories
- SQLite local database
- Responsive UI with Bootstrap

## Running Locally
App will run on `http://localhost:5000`.

## Deploying to Render
1. Push this code to a GitHub repo.
2. On Render, create a **new Web Service** from the repo.
3. Use:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python app.py`
4. Deploy.
